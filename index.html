Preguntas Frecuentes

Windows 11 tiene toda la potencia y seguridad de Windows 10, rediseñado visualmente. Así mismo, viene con nuevas herramientas, sonidos y aplicaciones. Cada detalle ha sido considerado. Todo ha resultado para brindarte una experiencia nueva y refrescante en tu dispositivo.

Los dispositivos con Windows 11 preinstalado estarán disponibles para la venta a final de este año.

Si tus accesorios funcionan con un dispositivo Windows 10 y tienen los requerimientos para Windows 11, deberían funcionar con normalidad Windows 11. De ser necesario, se puede revisar la información del empaque del fabricante para mayor información.

La mayoría de los dispositivos Windows 10 que están en el mercado actualmente estarán disponible para la actualización a Windows 11. Para que un dispositivo Windows 10 sea actualizado a Windows 11, este debe tener un mínimo de especificaciones técnicas requeridas para el funcionamiento de ciertas aplicaciones. El plan de actualización está siendo finalizado actualmente pero se considera que podría ponerse en marcha a finales del año 2021 y continuar en el año 2022. El tiempo específico varía según el dispositivo.

Si quisiera revisar si su dispositivo tiene el mínimo de requerimientos, puede descargar la aplicación PC Health Check para mayor información.

Es gratis. Pero únicamente los dispositivos que están utilizando la última versión de Windows 10 y que tengan los requerimientos mínimos estarán disponibles para la actualización. Puedes revisar si tienes las ultimas actualizaciones de Windows 10 en Configuración/Ver historial de actualizaciones
